# 天文图像增强功能说明

## 概述

新增了多种天文领域专用的图像增强算法，为不同类型的观测目标提供优化的增强效果。

## 增强算法

### 1. 基础增强 (BASIC)
- **描述**：直方图均衡化 + 锐化
- **原理**：
  - 在HSV色彩空间中对亮度通道进行直方图均衡化
  - 应用3x3锐化卷积核
- **适用场景**：一般天文图像
- **效果**：提高对比度和细节

### 2. CLAHE自适应直方图均衡化 (CLAHE)
- **描述**：自适应直方图均衡化
- **原理**：将图像分成小块，对每块独立均衡化
- **适用场景**：增强低对比度区域
- **效果**：保留局部细节，避免过度增强

### 3. 反锐化掩码 (UNSHARP_MASK)
- **描述**：基于高斯模糊的边缘增强
- **原理**：
  1. 对原图进行高斯模糊
  2. 计算原图与模糊图的差值（掩码）
  3. 将增强后的掩码加回原图
- **适用场景**：星点增强、减少模糊
- **效果**：增强边缘和细节

### 4. 形态学增强 (MORPHOLOGY)
- **描述**：基于形态学变换的增强
- **原理**：
  - 闭运算：连接断裂的星点
  - 顶帽变换：突出亮点
- **适用场景**：星点检测、去除小噪点
- **效果**：突出亮点，连接断裂星点

### 5. 降噪增强 (DENOISE)
- **描述**：双边滤波降噪
- **原理**：保留边缘的平滑滤波器
- **参数**：
  - d=9：滤波孔径直径
  - sigmaColor=75：颜色空间标准差
  - sigmaSpace=75：坐标空间标准差
- **适用场景**：高增益图像
- **效果**：降噪同时保留边缘

### 6. Lucy-Richardson反卷积 (LUCY_RICHARDSON)
- **描述**：盲反卷积恢复模糊
- **原理**：迭代恢复点扩散函数
- **适用场景**：大气模糊图像
- **效果**：恢复模糊细节

### 7. 线性拉伸 (STRETCHING)
- **描述**：扩展动态范围
- **原理**：将像素值拉伸到全范围
- **适用场景**：深空天体
- **效果**：扩展动态范围

### 8. Gamma校正 (GAMMA_CORRECTION)
- **描述**：非线性亮度调整
- **原理**：v' = (v/255)^(1/γ) * 255
- **参数**：
  - γ > 1：图像变暗
  - γ < 1：图像变亮
- **适用场景**：调整对比度和亮度
- **效果**：非线性亮度调整

### 9. 综合增强 (COMBINED)
- **描述**：组合多种算法
- **处理流程**：
  1. 降噪（双边滤波）
  2. CLAHE增强
  3. 反锐化掩码
  4. 形态学增强
  5. Gamma校正（γ=1.1）
- **适用场景**：大多数天文观测
- **效果**：最佳综合效果

## 专用增强模式

### 星点增强 (Enhance Stars)
- **描述**：专门针对星星的增强
- **处理流程**：
  1. 形态学顶帽变换提取星点
  2. 增强星点亮度
  3. 添加回原图（保护亮像素）
- **适用场景**：观测星星、恒星场
- **效果**：突出星点，提高可见性

### 深空天体增强 (Enhance Deep Sky)
- **描述**：专门针对星云、星系的增强
- **处理流程**：
  1. 降噪
  2. CLAHE增强低对比度细节
  3. 线性拉伸扩展动态范围
  4. 反锐化掩码增强细节
  5. Gamma校正（γ=0.9，使图像更亮）
- **适用场景**：星云、星系、星团
- **效果**：增强暗弱细节，提高对比度

### 行星表面增强 (Enhance Planetary)
- **描述**：专门针对月球、行星表面细节的增强
- **处理流程**：
  1. 反锐化掩码（行星需要锐化）
  2. 形态学增强
  3. 锐化
- **适用场景**：月球、木星、土星、火星
- **效果**：增强表面细节，提高清晰度

## 使用方法

### 界面操作

1. **选择增强算法**
   - 在"图像增强控制"面板中，从下拉菜单选择算法
   - 查看算法说明文字了解效果

2. **GPU加速选项**
   - 勾选"使用GPU加速"复选框
   - 注意：当前版本GPU功能尚未实现，将使用CPU处理

3. **应用增强**
   - 点击"应用增强"按钮，使用选择的算法
   - 或使用快捷键 E 快速增强

4. **专用增强**
   - 点击"星点增强"：专门增强星星
   - 点击"深空增强"：专门增强星云、星系
   - 点击"行星增强"：专门增强行星、月球

### 快捷键

- **E**：应用当前选择的增强算法

## 使用建议

### 月球观测
- 推荐算法：行星增强或综合增强
- 或使用：反锐化掩码 + 形态学增强
- 参数：短曝光，避免过曝

### 行星观测
- 推荐算法：行星增强
- 或使用：反锐化掩码 + 形态学增强
- 技巧：先增强，再叠加多帧

### 星星观测
- 推荐算法：星点增强
- 或使用：反锐化掩码 + 形态学增强
- 技巧：调整检测阈值

### 深空天体观测
- 推荐算法：深空天体增强或综合增强
- 或使用：降噪 + CLAHE + 线性拉伸
- 技巧：多帧叠加后再增强

### 高增益图像
- 推荐算法：降噪增强 + CLAHE
- 技巧：先降噪，再增强细节

### 低对比度图像
- 推荐算法：CLAHE或综合增强
- 技巧：使用线性拉伸扩展动态范围

## 算法组合建议

### 星云图像

### 星点图像

## 技术细节

### 直方图均衡化
```java
// HSV色彩空间分离
Mat hsv = cvtColor(image, COLOR_BGR2HSV);
MatVector channels = split(hsv);

// 只对V通道均衡化
equalizeHist(channels.get(2), channels.get(2));

// 合并通道
merge(channels, hsv);
image = cvtColor(hsv, COLOR_HSV2BGR);
```

### 反锐化掩码
```java
// 高斯模糊
Mat blurred = GaussianBlur(image, kernelSize=0, sigma=3);

// 计算掩码
Mat mask = image - blurred;

// 增强掩码
Mat enhancedMask = mask * 2.5;

// 添加回原图
Mat result = image + enhancedMask;
```

### 双边滤波
```java
// 双边滤波
Mat result = bilateralFilter(image,
    d=9,              // 滤波孔径
    sigmaColor=75,     // 颜色空间标准差
    sigmaSpace=75);    // 坐标空间标准差
```
