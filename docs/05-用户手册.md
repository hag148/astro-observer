## 1. 快速入门

### 1.1 启动程序

**Windows:**
1. 双击 `run.bat`
2. 或在命令行执行 `run.bat`

**Linux/Mac:**
1. 执行 `./run.sh`
2. 或执行 `java -jar target/astro-observer-1.0.0.jar`

### 1.2 首次使用流程

```
1. 选择摄像头
   ↓
2. 启动实时预览
   ↓
3. 调整参数
   ↓
4. 捕获图像
   ↓
5. 保存结果
```

### 1.3 最简操作（3步观测）

1. **选择摄像头**: 从下拉菜单选择您的 USB 摄像头
2. **点击启动**: 开始实时预览
3. **点击截图**: 捕获图像

就这么简单！

### 2.2 工具栏说明

| 组件 | 功能 |
|------|------|
| 摄像头选择 | 选择要使用的 USB 摄像头 |
| 分辨率 | 设置图像分辨率（支持 2K/FHD/HD/VGA） |
| 天文预设 | 快速应用天文观测参数 |
| 启动 | 开始实时预览 |
| 停止 | 停止预览 |
| 新会话 | 创建新的观测会话 |

### 2.3 参数控制

#### 曝光时间
- **范围**: 10ms - 1000ms
- **作用**: 控制图像的曝光时间
- **使用建议**:
  - 明亮目标（太阳、月球）：短曝光（10-50ms）
  - 行星：中等曝光（100-200ms）
  - 深空天体：长曝光（200-1000ms）

#### 增益
- **范围**: 0 - 100
- **作用**: 放大信号，提高亮度
- **使用建议**:
  - 明亮目标：低增益（0-30）
  - 一般目标：中等增益（30-60）
  - 暗弱目标：高增益（60-100）
  - **注意**: 增益过高会增加噪点

#### 亮度
- **范围**: 0 - 100
- **作用**: 调整图像整体亮度
- **使用建议**:
  - 根据实际观测效果调整
  - 建议配合曝光和增益使用

## 3. 基本操作

### 3.1 选择摄像头

1. 启动程序后，点击工具栏的"摄像头"下拉菜单
2. 选择您的 USB 摄像头
3. 系统会自动识别 UGREEN FHD 2K 摄像头

**提示**: 如果有多个摄像头，可以通过名称区分。

### 3.2 设置分辨率

1. 点击"分辨率"下拉菜单
2. 选择合适的分辨率
3. 推荐设置:
   - 一般观测: FHD (1920x1080)
   - 高质量观测: 2K (2560x1440)
   - 快速预览: HD (1280x720)

### 3.3 启动实时预览

1. 点击工具栏的"启动"按钮
2. 实时画面将显示在左侧
3. 状态栏显示"运行中"

### 3.4 捕获图像

**单张捕获**:
1. 确保摄像头正在运行
2. 点击"截图"按钮
3. 图像自动保存到 `captures/` 目录

**文件命名**:
```
capture_OBS_20250303_223045_123.png
```

### 3.5 停止预览

1. 点击工具栏的"停止"按钮
2. 摄像头停止捕获
3. 状态栏显示"已停止"

## 4. 天文观测

### 4.1 使用天文预设

#### 预设类型

| 预设 | 适用目标 | 分辨率 | 曝光 | 增益 |
|------|----------|--------|------|------|
| 月球 | 月球 | FHD | 50ms | 30 |
| 行星 | 木星、土星、火星等 | FHD | 100ms | 50 |
| 深空天体 | 星云、星团 | 2K | 200ms | 100 |
| 恒星 | 恒星、星系 | FHD | 150ms | 75 |
| 太阳 | 太阳（需滤镜） | FHD | 10ms | 1000 |

#### 应用预设

1. 点击"天文预设"下拉菜单
2. 选择目标类型（如"月球"）
3. 系统自动应用推荐参数

### 4.2 月球观测

#### 准备工作
- [ ] 将摄像头连接到望远镜
- [ ] 对准月球
- [ ] 应用"月球"预设

#### 观测步骤
1. 调整望远镜焦距，直到月球清晰
2. 点击"启动"开始预览
3. 调整曝光避免过曝
4. 点击"截图"捕获图像
5. 可选：点击"增强图像"提高细节

#### 拍摄技巧
- **曝光**: 避免月球边缘过曝
- **对焦**: 月球边缘细节清晰
- **时机**: 满月时最佳，新月适合观测环形山

### 4.3 行星观测

#### 可观测行星

| 行星 | 最佳观测时间 | 特点 |
|------|--------------|------|
| 木星 | 冬季 | 条纹、大红斑、卫星 |
| 土星 | 秋季 | 光环、卫星 |
| 火星 | 冲日前后 | 表面细节 |
| 金星 | 昏/晨 | 盈亏相位 |
| 水星 | 特定时期 | 盈亏相位 |

#### 观测步骤
1. 选择"行星"预设
2. 对准目标行星
3. 调整对焦
4. 连续拍摄多张图像（建议 50-100 张）
5. 使用"图像叠加"功能

#### 拍摄技巧
- **高帧率**: 连续拍摄视频帧
- **图像叠加**: 提高信噪比
- **等待大气稳定**: 日落后 1-2 小时

### 4.4 深空天体观测

#### 可观测目标

| 类型 | 代表目标 | 特点 |
|------|----------|------|
| 星云 | 猎户座大星云 | 气体云 |
| 星团 | M13 球状星团 | 恒星聚集 |
| 星系 | 仙女座星系 | 河外星系 |

#### 观测步骤
1. 选择"深空天体"预设
2. 使用 2K 分辨率
3. 长曝光（200-1000ms）
4. 拍摄大量图像（100+）
5. 使用 Kappa-Sigma 叠加

#### 拍摄技巧
- **远离光污染**: 到黑暗地区
- **长曝光**: 获取更多光子
- **多帧叠加**: 大幅提高信噪比
- **图像增强**: 提高对比度

### 4.5 恒星观测

#### 观测目标
- 明亮恒星（天狼星、织女星）
- 双星系统
- 星团
- 恒星云

#### 观测技巧
- 短曝光，保持星点锐利
- 中等分辨率
- 对焦到星点最小

### 4.6 太阳观测（危险！）

**⚠️ 重要安全提示**:
- **必须使用太阳滤镜**
- 严禁直接观测太阳
- 检查滤镜完整性

#### 观测步骤
1. 安装太阳滤镜
2. 选择"太阳"预设
3. 极短曝光（10ms）
4. 调整对焦
5. 捕获图像

#### 可观测现象
- 太阳黑子
- 日珥（需专用滤镜）
- 日面细节

## 5. 图像处理

### 5.1 图像增强

#### 功能说明
- 直方图均衡化
- 对比度增强
- 锐化处理
- 亮度调整

#### 使用方法
1. 捕获或加载图像
2. 点击"增强图像"按钮
3. 等待处理完成
4. 查看增强效果

#### 效果对比
```
原始图像 → 图像增强 → 细节更清晰，对比度提高
```

### 5.2 图像叠加

#### 叠加方法

| 方法 | 速度 | 质量 | 适用场景 |
|------|------|------|----------|
| 平均叠加 | 快 | 一般 | 日常使用 |
| 中位数叠加 | 中等 | 好 | 有干扰时 |
| Kappa-Sigma | 慢 | 最佳 | 高质量需求 |

#### 使用方法

**自动叠加**:
1. 连续拍摄多张图像
2. 系统自动添加到堆栈
3. 点击"图像叠加"
4. 选择叠加方法

**手动叠加**:
1. 先拍摄多张图像
2. 使用"图像叠加"功能
3. 查看叠加结果

#### 叠加效果
- **信噪比提升**: √N（N 为帧数）
- **100 帧叠加**: 信噪比提升 10 倍
- **减少噪点**: 突出暗弱细节

### 5.3 天体检测

#### 功能说明
- 自动识别星星
- 计算亮度
- 测量位置

#### 使用方法
1. 捕获图像
2. 点击"检测天体"按钮
3. 查看检测结果

#### 检测参数
- **阈值**: 控制检测灵敏度（建议 50-100）
- **面积过滤**: 忽略小噪点

## 6. 高级功能

### 6.1 观测会话

#### 创建会话
1. 点击工具栏"新会话"按钮
2. 输入观测目标
3. 记录观测条件

#### 会话信息
- 会话 ID
- 观测目标
- 开始/结束时间
- 图像数量
- 观测时长
- 备注

#### 保存会话
1. 在备注区域记录信息
2. 点击"保存会话"按钮
3. 信息保存到 `sessions/` 目录

### 6.2 批量拍摄

#### 自动拍摄脚本
1. 使用天文预设
2. 启动实时预览
3. 连续点击"截图"按钮
4. 系统自动编号保存

#### 时间间隔拍摄
```
手动控制拍摄间隔，建议 1-2 秒/张
```

### 6.3 数据管理

#### 图像存储
- **目录**: `captures/`
- **命名**: `{prefix}_{timestamp}_{random}.png`
- **分类**: 通过前缀区分不同观测

#### 会话记录
- **目录**: `sessions/`
- **格式**: `{session_id}.txt`
- **内容**: 观测信息、参数、备注

## 7. 常见问题

### 7.1 摄像头相关

**Q: 摄像头未识别？**
A:
1. 检查 USB 连接
2. 确认驱动已安装
3. 尝试重新插拔
4. 重启程序

**Q: 图像模糊？**
A:
1. 调整望远镜焦距
2. 等待大气稳定
3. 使用高分辨率

**Q: 分辨率无法选择？**
A:
1. 摄像头可能不支持该分辨率
2. 选择其他可用分辨率
3. 检查 USB 传输速度

### 7.2 图像处理

**Q: 检测不到天体？**
A:
1. 确保图像足够亮
2. 先进行图像叠加
3. 调整检测阈值
4. 尝试图像增强

**Q: 叠加后图像变亮？**
A: 正常现象。叠加会提高信噪比，可以调整亮度。

**Q: 增强后图像失真？**
A:
1. 减少增强强度
2. 使用原始图像
3. 调整曝光参数

### 7.3 性能问题

**Q: 帧率过低？**
A:
1. 降低分辨率
2. 关闭其他应用
3. 使用 USB 3.0
4. 增加内存

**Q: 内存不足？**
A:
1. 减少图像堆栈大小
2. 定期清理 captures 目录
3. 增加系统内存

### 7.4 观测问题

**Q: 看不到目标？**
A:
1. 检查望远镜指向
2. 调整对焦
3. 增加曝光或增益
4. 等待大气稳定

**Q: 目标过曝？**
A:
1. 减少曝光时间
2. 减少增益
3. 使用滤镜

## 8. 技巧与窍门

### 8.1 提高观测质量

#### 大气稳定
- 最佳时间: 日后 1-2 小时
- 避免热气流干扰
- 等待风力减小

#### 对焦技巧
- 先对准明亮的恒星
- 调整到星点最小
- 保持对焦精度

#### 拍摄时机
- **月球**: 满月前后，上下弦时
- **行星**: 冲日前后最佳
- **深空**: 新月前后（天空最暗）

### 8.2 图像处理技巧

#### 叠加策略
- **行星**: 100-200 帧，平均叠加
- **深空**: 200-500 帧，Kappa-Sigma
- **月球**: 10-30 帧，中位数叠加

#### 增强建议
- 先叠加，再增强
- 适度增强，避免过度
- 保留原始图像

### 8.3 设备优化

#### 摄像头设置
- 选择高分辨率
- 降低 ISO（如果支持）
- 禁用自动功能

#### 望远镜设置
- 使用高倍率目镜
- 保持光轴准直
- 使用赤道仪跟踪

### 8.4 数据管理

#### 命名规则
```
{目标}_{日期}_{类型}_{编号}
例如: moon_20250303_raw_001.png
```

#### 备份建议
- 定期备份重要图像
- 使用外部存储
- 云端备份（可选）
