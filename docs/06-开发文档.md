# 天文观测系统 Pro - 开发文档

## 1. 开发环境搭建

### 1.1 IDE 配置

#### IntelliJ IDEA
1. **导入项目**
   - File → Open → 选择项目根目录
   - 选择 Maven 项目
   - 等待依赖下载完成

2. **配置 SDK**
   - File → Project Structure → Project SDK
   - 选择 JDK 17 或更高版本

3. **代码风格**
   - File → Settings → Editor → Code Style
   - 导入配置文件

#### Eclipse
1. **导入项目**
   - File → Import → Maven → Existing Maven Projects
   - 选择项目根目录
   - 完成导入

2. **配置 JRE**
   - Window → Preferences → Java → Installed JREs
   - 添加 JDK 17

#### VS Code
1. **安装扩展**
   - Extension Pack for Java
   - Maven for Java

2. **打开项目**
   - File → Open Folder
   - 选择项目根目录

### 1.2 依赖管理

#### Maven 仓库配置
```xml
<repositories>
    <repository>
        <id>central</id>
        <url>https://repo.maven.apache.org/maven2</url>
    </repository>
</repositories>
```

### 1.3 编译运行

#### 命令行编译
```bash
mvn clean compile
mvn package
mvn package -DskipTests
```

## 2. 代码规范

### 2.1 命名规范
- 类名：PascalCase
- 方法名：camelCase
- 常量：UPPER_SNAKE_CASE
- 变量：camelCase

### 2.2 注释规范
- 类级注释
- 方法级注释
- 参数说明
- 返回值说明

### 2.3 代码格式化
- 4 空格缩进
- 最大 120 字符/行
- 类之间 2 空行
- 方法之间 1 空行

## 3. 架构设计

### 3.1 分层架构
- Presentation Layer (GUI)
- Business Layer (Camera, Image, Config)
- Integration Layer (WebCam, OpenCV)
- Hardware Layer (Camera, Telescope)

### 3.2 模块依赖
```
GUI Module
    ↓ depends on
Business Modules
    ↓ depends on
Integration Libraries
```

### 3.3 设计模式
- 观察者模式：摄像头帧监听
- 工厂模式：摄像头配置创建
- 策略模式：图像叠加算法
- 单例模式：OpenCV 库加载
