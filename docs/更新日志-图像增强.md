## 🆕 新增功能

### 1. 天文图像增强处理器 (AstronomyImageEnhancer.java)

#### 1.1 增强算法（9种）

| 算法 | 中文名称 | 适用场景 | 处理原理 |
|------|---------|----------|----------|
| BASIC | 基础增强 | 一般天文图像 | HSV直方图均衡化 + 3x3锐化核 |
| CLAHE | 自适应直方图均衡化 | 低对比度区域 | 分块均衡化，保留局部细节 |
| UNSHARP_MASK | 反锐化掩码 | 星点增强、去模糊 | 原图 - 高斯模糊 = 边缘 |
| MORPHOLOGY | 形态学增强 | 星点检测 | 闭运算 + 顶帽变换 |
| DENOISE | 降噪增强 | 高增益图像 | 双边滤波（保留边缘）|
| LUCY_RICHARDSON | Lucy-Richardson反卷积 | 大气模糊图像 | 简化版：维纳滤波 |
| STRETCHING | 线性拉伸 | 深空天体 | 动态范围扩展 |
| GAMMA_CORRECTION | Gamma校正 | 亮度/对比度调整 | 非线性亮度调整 |
| COMBINED | 综合增强 | 最佳效果 | 5步组合处理 |

#### 1.2 专用增强模式（3种）

**星点增强 (Enhance Stars)**
- 专门针对星星观测
- 处理流程：形态学顶帽变换 → 亮度增强 → 添加回原图
- 保护亮像素，只增强暗部星点

**深空天体增强 (Enhance Deep Sky)**
- 专门针对星云、星系、星团
- 处理流程：降噪 → CLAHE → 线性拉伸 → 反锐化掩码 → Gamma(0.9)
- 提高暗弱细节，扩展动态范围

**行星表面增强 (Enhance Planetary)**
- 专门针对月球、行星表面
- 处理流程：反锐化掩码 → 形态学增强 → 锐化
- 增强表面细节，提高清晰度

## 🔧 技术实现

### 1. OpenCV API兼容性修复

**问题**：OpenCV 4.13.0 API变更

**修复内容**：

#### 1.1 multiply() 方法
```java
// 错误写法
opencv_core.multiply(mat, new Scalar(2.5), result);

// 正确写法
Mat scalarMat = new Mat(mat.rows(), mat.cols(), mat.type(), new Scalar(2.5));
opencv_core.multiply(mat, scalarMat, result);
scalarMat.release();
```

#### 1.2 convertTo() 方法
```java
// 错误写法
opencv_core.convertTo(mat, result, -1, alpha, beta);

// 正确写法（分两步）
Mat scalarScale = new Mat(mat.rows(), mat.cols(), mat.type(), new Scalar(alpha));
Mat scaled = new Mat();
opencv_core.multiply(mat, scalarScale, scaled);
scalarScale.release();

Mat scalarOffset = new Mat(scaled.rows(), scaled.cols(), scaled.type(), new Scalar(beta));
opencv_core.add(scaled, scalarOffset, result);
scalarOffset.release();
scaled.release();
```

### 2. 算法原理详解

#### 2.1 基础增强 (BASIC)
```java
// HSV转换
Mat hsv = cvtColor(image, COLOR_BGR2HSV);
MatVector channels = split(hsv);

// 只均衡化V（亮度）通道
equalizeHist(channels.get(2), channels.get(2));

// 合并并转换回BGR
merge(channels, hsv);
Mat enhanced = cvtColor(hsv, COLOR_HSV2BGR);

// 锐化
Mat kernel = [0, -1, 0; -1, 5, -1; 0, -1, 0];
filter2D(enhanced, enhanced, kernel);
```

#### 2.2 反锐化掩码 (UNSHARP_MASK)
```java
// 1. 高斯模糊
Mat blurred = GaussianBlur(image, size=(0,0), sigma=3);

// 2. 计算掩码
Mat mask = image - blurred;

// 3. 增强掩码
Mat scalarMat = createScalarMat(mask, 2.5);
Mat enhancedMask = multiply(mask, scalarMat);

// 4. 添加回原图
Mat result = image + enhancedMask;
```

#### 2.3 双边滤波 (DENOISE)
```java
// 双边滤波：考虑空间距离和颜色差异
Mat result = bilateralFilter(image,
    d=9,          // 滤波孔径
    sigmaColor=75,  // 颜色空间标准差
    sigmaSpace=75); // 坐标空间标准差
```

#### 2.4 形态学顶帽变换 (MORPHOLOGY)
```java
// 1. 闭运算（连接断裂）
Mat closed = morphologyEx(gray, MORPH_CLOSE, kernel);

// 2. 顶帽变换（提取亮点）
Mat tophat = morphologyEx(closed, MORPH_TOPHAT, kernel);

// 3. 添加回原图
Mat result = gray + tophat;
```

## 📝 使用建议

### 月球观测
- **推荐**：行星增强或综合增强
- **参数**：短曝光（10-50ms），低增益（0-30）
- **技巧**：避免过曝，关注边缘细节

### 行星观测
- **推荐**：行星增强
- **参数**：中等曝光（100-200ms），中等增益（30-60）
- **技巧**：多帧叠加后再增强

### 星星观测
- **推荐**：星点增强
- **参数**：中等曝光（100-150ms），中等增益（30-60）
- **技巧**：调整检测阈值

### 深空天体观测
- **推荐**：深空天体增强或综合增强
- **参数**：长曝光（200-1000ms），高增益（60-100）
- **技巧**：多帧叠加（100+帧）后再增强

## ⚠️ 已知限制

### 1. GPU加速
- **状态**：已预留接口，未实现
- **原因**：需要OpenCV CUDA支持
- **计划**：未来版本添加

### 2. Lucy-Richardson
- **状态**：简化版实现
- **原因**：完整的反卷积算法复杂
- **效果**：可处理轻度模糊，重度模糊效果有限

### 3. CLAHE
- **状态**：使用标准均衡化替代
- **原因**：OpenCV 4.13.0限制
- **效果**：功能类似，但不是真正的自适应
