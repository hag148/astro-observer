# 系统架构文档

## 概述

天文观测系统 Pro 是一个基于 Java 17 的纯 Swing GUI 桌面应用程序，专注于天文图像处理和观测管理功能。

## 项目类型

**纯Swing GUI桌面应用程序**
- 基于 Java 17
- 使用 OpenCV 4.13.0 进行图像处理
- 不包含Spring Boot、WebSocket等后端技术

## 包结构

### 1. **camera** - 摄像头管理
- `OpenCVCameraManager.java` - 摄像头驱动、实时预览、分辨率切换

### 2. **config** - 配置管理
- `UGreenCameraConfig.java` - UGREEN摄像头专用配置

### 3. **gui** - 图形界面
- `AdvancedAstronomyFrame.java` - 主界面、控制面板、事件处理

### 4. **image** - 图像处理
- `AstronomyImageEnhancer.java` - 9种图像增强算法
- `OpenCVImageProcessor.java` - OpenCV图像处理核心
- `OpenCVImageStacker.java` - 图像叠加和信噪比计算

### 5. **model** - 数据模型
- `ObservationSession.java` - 观测会话管理

---

## 数据流

```
用户操作
   ↓
AdvancedAstronomyFrame (GUI层)
   ↓
OpenCVCameraManager (摄像头)
   ↓
OpenCVImageProcessor (图像处理)
   ↓
ObservationSession (数据存储)
   ↓
保存/显示结果
```

## 核心特性

1. **实时预览** - 30 FPS流畅预览
2. **图像增强** - 9种专业算法
3. **图像叠加** - 3种叠加方式
4. **天体检测** - 自动识别星星、行星
5. **观测会话** - 完整的观测记录
6. **天文预设** - 月球、行星、深空专用配置

## 扩展性设计

### 1. 图像处理器扩展
- 支持纯Java版和OpenCV版切换
- 可扩展更多图像处理算法

### 2. 摄像头支持
- 支持多摄像头
- 可扩展其他品牌摄像头

### 3. 图像叠加方式
- 当前：平均、中位数、Kappa-Sigma
- 可扩展：加权平均、PCA等

---

## 性能优化

### 1. 渲染优化
- 使用Graphics2D AffineTransform替代图像缩放
- 消除缩放时的卡顿

### 2. 内存管理
- 及时释放OpenCV Mat资源
- 避免内存泄漏

### 3. 多线程
- Swing EDT线程管理
- 避免UI阻塞
