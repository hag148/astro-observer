# 天文观测系统 Pro - 项目概述

## 1. 项目简介

### 1.1 项目背景
天文观测系统 Pro 是一个基于 Java 和 USB 摄像头的专业天文观测应用程序。该系统专为业余天文学者和摄影爱好者设计，通过普通 USB 摄像头配合望远镜，实现月球、行星、恒星和深空天体的观测与摄影。

### 1.2 项目目标
- 提供易于使用的天文观测工具
- 支持多种 USB 摄像头设备
- 实现专业级的图像处理和叠加功能
- 提供完整的天文观测工作流

### 1.3 目标用户
- 业余天文爱好者
- 天文摄影初学者
- 教育机构的天文教学
- 科学研究辅助观测

## 2. 核心功能

### 2.1 摄像头管理
- 自动检测 USB 摄像头
- 多摄像头切换支持
- 分辨率动态调整
- 实时视频流预览

### 2.2 图像采集
- 单帧图像捕获
- 连续拍摄支持
- 时间间隔拍摄
- RAW 格式支持

### 2.3 图像处理
- 天文专用图像增强
- 直方图均衡化
- 锐化处理
- 降噪处理

### 2.4 图像叠加
- 平均叠加
- 中位数叠加
- Kappa-Sigma 叠加
- 信噪比计算

### 2.5 天体检测
- 自动星点识别
- 行星检测
- 亮度测量
- 位置标记

### 2.6 观测管理
- 观测会话记录
- 图像分类存储
- 观测日志
- 数据导出

### 3.3 模块划分

#### 3.3.1 摄像头模块 (camera)
- **CameraManager**: 摄像头设备管理
- 功能：摄像头检测、连接控制、参数调节

#### 3.3.2 图像处理模块 (image)
- **ImageProcessor**: 图像处理核心
- **ImageStacker**: 图像叠加处理
- 功能：图像增强、天体检测、叠加算法

#### 3.3.3 配置模块 (config)
- **UGreenCameraConfig**: 摄像头专用配置
- 功能：预设参数、分辨率管理

#### 3.3.4 数据模型 (model)
- **ObservationSession**: 观测会话模型
- 功能：会话数据管理

#### 3.3.5 用户界面 (gui)
- **AdvancedAstronomyFrame**: 主界面
- 功能：用户交互、操作控制
