# 天文观测系统 Pro - 更新日志

## [1.0.0] - 2025-03-03

### 新增功能
- 🎉 项目初始发布
- ✅ USB 摄像头实时预览
- ✅ 多摄像头支持
- ✅ 可调节分辨率（支持 2K/FHD/HD/VGA）
- ✅ 图像捕获和保存
- ✅ 天文预设配置（月球、行星、深空天体、恒星、太阳）
- ✅ 图像增强处理（直方图均衡化、锐化）
- ✅ 图像叠加功能（平均、中位数、Kappa-Sigma）
- ✅ 天体自动检测
- ✅ 观测会话管理
- ✅ 完整的 GUI 界面
- ✅ UGREEN FHD 2K 摄像头优化

### 技术特性
- 基于 Java 17
- 使用 OpenCV 进行图像处理
- WebCam Capture API 摄像头访问
- FlatLaf 现代化 UI 主题
- SLF4J 日志框架

## 版本说明

### 版本号规则
遵循语义化版本控制 (SemVer):
- MAJOR.MINOR.PATCH
  - MAJOR: 不兼容的 API 变更
  - MINOR: 向下兼容的新功能
  - PATCH: 向下兼容的问题修复

### 更新类型
- 新增功能 ✅
- 问题修复 🐛
- 性能优化 ⚡
- 文档更新 📚
- 安全修复 🔒
