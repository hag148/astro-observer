# 天文观测系统 Pro - 更新日志

## [1.1.0] - 2026-03-04

### 新增功能
- ✅ 实时画面天体标识：点击检测出的天体信息时在画面上显示标记
- ✅ 手动对焦控制：支持通过滑块和数值输入框精确控制摄像头对焦
- ✅ 点击放大：点击画面任意位置进行放大查看
- ✅ 地图式交互：支持拖拽移动、滚轮缩放、双击定位
- ✅ 自动对焦切换：支持启用/禁用自动对焦

### 问题修复 🐛
- 🐛 修复画面闪烁和UI字体消失问题（Swing线程优化）
- 🐛 修复中文字体显示乱码（使用Microsoft YaHei字体）

### 性能优化 ⚡
- ⚡ 渲染性能大幅提升：使用Graphics2D AffineTransform替代图像缩放
- ⚡ 缩放操作流畅性优化：消除缩放时的卡顿和延迟
- ⚡ 内存使用优化：缩放时不再创建大尺寸图像副本

### 技术改进
- 确保所有UI更新在EDT线程执行
- 优化渲染管线，支持平滑的视图变换
- 增强坐标转换逻辑，实现精确的屏幕-图像坐标映射

## [1.0.0] - 2025-03-03

### 新增功能
- 🎉 项目初始发布
- ✅ USB 摄像头实时预览
- ✅ 多摄像头支持
- ✅ 可调节分辨率（支持 2K/FHD/HD/VGA）
- ✅ 图像捕获和保存
- ✅ 天文预设配置（月球、行星、深空天体、恒星、太阳）
- ✅ 图像增强处理（直方图均衡化、锐化）
- ✅ 图像叠加功能（平均、中位数、Kappa-Sigma）
- ✅ 天体自动检测
- ✅ 观测会话管理
- ✅ 完整的 GUI 界面
- ✅ UGREEN FHD 2K 摄像头优化

### 技术特性
- 基于 Java 17
- 使用 OpenCV 进行图像处理
- WebCam Capture API 摄像头访问
- FlatLaf 现代化 UI 主题
- SLF4J 日志框架

## 版本说明

### 版本号规则
遵循语义化版本控制 (SemVer):
- MAJOR.MINOR.PATCH
  - MAJOR: 不兼容的 API 变更
  - MINOR: 向下兼容的新功能
  - PATCH: 向下兼容的问题修复

### 更新类型
- 新增功能 ✅
- 问题修复 🐛
- 性能优化 ⚡
- 文档更新 📚
- 安全修复 🔒
