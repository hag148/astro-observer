# 天文观测系统 Pro (Astronomy Observer Pro)

基于Java和USB摄像头的专业天文观测应用程序，专为UGREEN FHD 2K摄像头优化。

## 主要功能

### 核心功能
- 实时摄像头预览
- 多摄像头支持
- 可调节分辨率（支持2K/2560x1440）
- 高速图像捕获
- 图像保存和管理

### 天文观测专用功能
- **天文预设**：月球、行星、深空天体、恒星、太阳观测优化
- **图像叠加**：支持平均叠加、中位数叠加、Kappa-Sigma叠加
- **图像增强**：直方图均衡化、锐化、对比度调整
- **天体检测**：自动识别星星、行星等天体对象
- **观测会话**：完整的观测记录管理
- **天体标识**：点击检测出的天体在实时画面显示标记
- **交互式观测**：支持地图式操作（拖拽、缩放、定位）

### 摄像头控制
- 曝光时间调节
- 增益控制
- 亮度/对比度调整
- 分辨率选择
- 手动/自动对焦控制
- 针对UGREEN摄像头的优化配置

### 界面交互
- 点击放大：点击画面任意位置进行放大查看
- 拖拽移动：拖拽画面移动查看区域
- 滚轮缩放：使用鼠标滚轮平滑缩放
- 双击定位：双击快速定位到指定位置
- 高性能渲染：流畅的缩放和移动体验

## 系统要求

- Java 17 或更高版本
- Maven 3.6+
- USB摄像头（推荐 UGREEN FHD 2K）

## 使用说明

### 1. 选择摄像头
- 从顶部工具栏选择您的USB摄像头
- 系统会自动识别UGREEN FHD 2K摄像头

### 2. 应用天文预设
- 从预设下拉菜单选择观测目标：
  - 月球：短曝光，低增益
  - 行星：中等曝光，中等增益
  - 深空天体：长曝光，高增益
  - 恒星：中等曝光，中等增益
  - 太阳：极短曝光（需配太阳滤镜）

### 3. 调整参数
- **曝光时间**：调整曝光时长（10-1000ms）
- **增益**：调整信号放大（0-100）
- **亮度**：调整图像亮度（0-100）
- **对焦**：手动调整对焦或启用自动对焦

### 4. 启动观测
- 点击"启动"按钮开始实时预览
- 使用"截图"捕获单张图像
- 使用"图像叠加"进行多帧叠加

### 5. 交互式查看
- **点击放大**：点击画面任意位置放大查看细节
- **拖拽移动**：按住鼠标左键拖拽画面移动查看区域
- **滚轮缩放**：使用鼠标滚轮平滑缩放画面
- **双击定位**：双击快速定位到指定位置
- **天体标识**：点击天体检测结果中的某一项，画面会显示黄色圆圈和红色十字准星标记

### 6. 图像处理
- **增强图像**：应用天文专用图像增强
- **检测天体**：自动识别图像中的天体对象
- **天体标记**：在检测到的天体位置显示标记

### 7. 观测会话管理
- 点击"新观测会话"创建新会话
- 在备注区域记录观测信息
- 点击"保存会话"保存观测数据

## 天文观测建议

### 设备配置
1. 将摄像头连接到望远镜目镜接口
2. 使用三脚架或赤道仪固定设备
3. 确保电源稳定

### 观测环境
1. 远离城市光污染
2. 新月前后天空最暗
3. 避免月光干扰（除观测月球外）
4. 等待大气稳定（约1-2小时后日落）

### 拍摄技巧
1. **行星**：使用高帧率视频，后期叠加
2. **月球**：短曝光，避免过曝
3. **深空天体**：长曝光，多帧叠加
4. **恒星**：中等曝光，保持星点锐利

### 图像叠加
1. 捕获至少30-100帧图像
2. 使用"图像叠加"功能处理
3. 叠加可大幅提高信噪比
4. 支持三种叠加方式：
   - 平均叠加：快速，适合大多数情况
   - 中位数叠加：去除异常值
   - Kappa-Sigma叠加：最佳质量

### 图像增强
1. 对叠加后的图像应用增强
2. 调整亮度/对比度突出细节
3. 使用直方图均衡化提高对比度
4. 适度锐化增强细节

## UGREEN FHD 2K 摄像头优化

### 支持的分辨率
- 2K QHD: 2560x1440
- FHD: 1920x1080
- HD: 1280x720
- VGA: 640x480

### 天文观测推荐配置
- **月球**：FHD，曝光50ms，增益30
- **行星**：FHD，曝光100ms，增益50
- **深空天体**：2K QHD，曝光200ms，增益100
- **恒星**：FHD，曝光150ms，增益75
- **太阳**：FHD，曝光10ms，增益1000（必须使用滤镜）

## 常见问题

### Q: 找不到摄像头？
A: 确保摄像头已正确连接，检查驱动程序是否安装，尝试重新插拔USB。

### Q: 检测不到天体？
A:
- 确保图像足够亮
- 先进行图像叠加提高信噪比
- 调整检测参数
- 尝试图像增强处理

### Q: 图像质量差？
A:
- 检查摄像头清洁度
- 尝试更高分辨率
- 在低光环境下使用增强功能
- 进行多帧叠加处理

### Q: 堆叠后图像变亮？
A: 这是正常的。平均叠加会保留信号并减少噪声，图像会更干净，但可能需要调整亮度。

### Q: 如何连接到望远镜？
A: 使用专用的望远镜摄像头转接环，将摄像头固定在望远镜目镜位置。
